What is web development?

Web development is the process of building websites and applications for the internet, or for a private network known as an intranet. Web development isn't exactly concerned with the design of a website; rather, it's all about the coding and programming that powers the website's functionality. From the most simple, static web pages to social media platforms and apps, from e-commerce websites to content management systems (CMS)—all the tools we use via the internet on a daily basis have been built by developers.
